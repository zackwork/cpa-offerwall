CPA OFFERWALL will display cpa marketing offers in a template as offerwall using your json link to monetize your traffic.

== Description ==

CPA OFFERWALL is a plugin which helps users display cpa offers on their wordpress website, using this plugin will display cpa offers by creating a page and using template we offer to import cpagrip offers to your website.

This plugin currently only supports cpagrip api and can only display cpagrip offers.

This plugin will help you promote cpagrip offers to your visitors/traffic and monetize them with premium offers and get extra revenue.

It's easy to install and uses light classic card template, i will add more templates in the future.


1. This plugin currently supports only `CPAGRIP Offerwalls` so if you don't have account go and register at [www.cpagrip.com](https://www.cpagrip.com/)
2. When you have your cpagrip account login to it and, from your dashboard go to `Offer Tools` in the side-navbar.
3. Below Offer Tools click on `JSON Offer Feed` there you will see `Your JSON Feed URL:` Copy the json link below it.
4. Go to Your Wordpress Dashboard and from right side-menu under setting click on CPA OW Settings, `You Need to have Installed and activated this plugin first` in this page you will see Json Url field Paste the json url you copied earlier from cpgrip and paste it there and then save click save.
5. After you follow above steps your the plugin is configured successfuly to use it to display the cpa offers into any page go and create new page, any page is fine, in the page creation page on the right side under Page Attributes you will see Templates Dropdown click there and choose Default CPA Template, That's the Only Template currently available that can display the offers in your page.
6. When You Select it choose a title for the page and description if you need and then publish the page, asfter that visit it.



== Installation ==


1. Install easily with the WordPress plugin control panel or manually download the plugin and upload the folder `cpa-offerwall` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
